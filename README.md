# Job Market Data Analysis

**Tech:** Python, Pandas, NumPy, Matplotlib/Seaborn

This project explores a **job market dataset** containing features like work experience, employment type, job title, salary, and currency. The HTML file demonstrates the full data analysis process and visualizations.

## Project Overview
- Performed **data cleaning and preparation**, handling missing values and ensuring consistency.  
- Conducted **exploratory data analysis (EDA)** to uncover trends and insights in salaries, job titles, and employment types.  
- Created **visualizations** to communicate patterns and key findings clearly.  
- Demonstrated skills in **data handling, analysis, and visualization** suitable for AI and data-driven projects.
